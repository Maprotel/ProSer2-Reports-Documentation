# MainStoredQueries

## msq_query_id - msq_consulta_id
* Número secuencial generado automáticamente por el sistema cada vez que se inserta un registro de consulta.

## msq_query_status - msq_consulta_estado
* Estado de cada consulta en A(activo), I(inactivo).

## msq_query_chk - msq_consulta_chk
* xxx

## msq_query_name - msq_consulta_nombre
* Nombre de cada consulta registrada.

## msq_query_selection - msq_consulta_seleccion
* Selección de cada consulta registrada.

## msq_query_response - msq_consulta_respuesta
* Respuesta de cada consulta.

## ms_query_rotation - msq_consulta_rotacion
* Rotacińde las consultas registradas.

## msq_query_datetime - msq_consulta_fecha_hora
* Fecha y hora de cada consulta.

## msq_query_date - msq_consulta_fecha
* Fecha de cada consulta registrada.

## msq_query_time - msq_consulta_tiempo
* Tiempo de cada consulta registrada.
