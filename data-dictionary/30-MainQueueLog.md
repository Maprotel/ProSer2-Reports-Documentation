# MainQueueLog

## queuelog_id - registro_cola_id
* Número secuencial generado automáticamente por el sistema cada vez que se inserta un registro de cola.

## queuelog_time - registro_cola_tiempo
* Tiempo de cada registro de cola.
	
## queuelog_uniqueid - registro_cola_unico_id
* Número único de resgistro de cola.

## queuelog_queuename - registro_cola_cola_nombre
* Nombre de cada regsitro de cola.

## queuelog_agent - registro_cola_agente
* Agente de cada registro de cola.

## queuelog_event - registro_cola_evento
* Evento de cada registro de cola.

## queuelog_data - registro_cola_datos
* Datos de cada resgistro de cola.

## queuelog_data1 - registro_cola_datos1
* Primeros datos de cada resgistro de cola.

## queuelog_data2 - registro_cola_datos2
* Segundos datos de cada resgistro de cola.

## queuelog_data3 - registro_cola_datos3
* Terceros datos de cada resgistro de cola.

## queuelog_data4 - registro_cola_datos4
* Cuartos datos de cada resgistro de cola.

## queuelog_data5 - registro_cola_datos4
* Quintos dats de cada resgistro de cola.

## queuelog_created - registro_cola_creado
* Registro de cada creacion de colas.

## queuelog_date - registro_cola_fecha
* Fecha de cada registro de cola.
